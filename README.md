## 명령어
- /spawn, /sp, /넴주
```
스폰포인트로 이동
```

- /tools, /t
```
[내구성I, 소실저주]이(가) 적용된 [돌검, 돌곡, 돌도끼]를 인벤토리에 추가
```

- /back, /b
```
죽기 전 마지막 위치로 이동
경고) 죽은 기록이 없는 상태에서 명령어를 치면 3초 후 사살
```
